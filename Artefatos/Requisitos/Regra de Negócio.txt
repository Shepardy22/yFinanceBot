Sumário:

O projeto consiste em desenvolver um software que permita ao usuário buscar informações e gráficos de ativos financeiros utilizando o Yahoo Finance. O sistema será desenvolvido em Python com o padrão de projeto MVC e utilizará as bibliotecas yfinance, matplotlib, tkinter, pandas e numpy. Além disso, o código será versionado com o gitFlow e ficará disponível no GitHub.

Descrição das Funções e Métodos:

Função buscar_dados(ativo, timeframe): Essa função irá buscar os dados e cotações do ativo escolhido no Yahoo Finance utilizando a biblioteca yfinance. Caso o ativo ou timeframe escolhido seja inválido, a função apresentará uma mensagem de erro.

Método plotar_grafico(dados): Esse método irá plotar o gráfico do ativo selecionado utilizando a biblioteca matplotlib.

Função exportar_grafico(nome_arquivo): Essa função irá exportar o gráfico como imagem em formato PNG utilizando a biblioteca matplotlib.

Função apresentar_mensagem_erro(mensagem): Essa função irá apresentar uma mensagem de erro na interface gráfica do programa.

Método atualizar_view(): Esse método irá atualizar a interface gráfica do programa após a busca de dados e plotagem do gráfico.

Descrição das Bibliotecas:

yfinance: biblioteca utilizada para buscar os dados e cotações do Yahoo Finance.

matplotlib: biblioteca utilizada para plotar os gráficos do ativo selecionado e exportá-los como imagem em formato PNG.

tkinter: biblioteca utilizada para criar a interface gráfica do programa.

pandas: biblioteca utilizada para manipulação de dados.

numpy: biblioteca utilizada para cálculos numéricos.

Descrição do Padrão de Projeto:

Será utilizado o padrão de projeto MVC (Model-View-Controller), que consiste em separar a aplicação em três partes: o Model, responsável pela manipulação dos dados; o View, responsável pela interface gráfica; e o Controller, responsável por controlar as interações entre o Model e o View. Dessa forma, o programa será mais organizado, escalável e de fácil manutenção.

Descrição do GitFlow:

Será utilizado o gitFlow para o versionamento do código e futuras atualizações no programa. O gitFlow é um modelo de fluxo de trabalho para o git que divide o desenvolvimento em duas branches principais: a branch master, que contém o código em produção; e a branch develop, que contém o código em desenvolvimento. Além disso, serão criadas outras branches para as features, hotfixes e releases, seguindo as melhores práticas de desenvolvimento de software.

Conclusão:

Com essa modelagem de funções, métodos, bibliotecas, padrão de projeto e gitFlow, a equipe de desenvolvimento terá um guia completo para implementação dos requisitos funcionais e não funcionais do software de busca e plotagem de dados financeiros utilizando o Yahoo Finance em Python.
